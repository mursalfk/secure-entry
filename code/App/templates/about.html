Problem Statement

Traditional door authentication methods can be easily bypassed, making home security a major concern. Our solution enhances security using facial and voice recognition.

Solution

We propose an intrusion alarm system using a smartphone installed at the door. Users will undergo face and voice verification before entry. Two failed attempts will trigger an alarm.

Objective

To develop a dual-authentication security system that enhances home security while providing users with a seamless experience.

Authentication Process

  • Users authenticate via a smartphone.
  • Face and voice are captured for verification.

Dual Verification

  • Face recognition using OpenCV.
  • Voice authentication using pre-recorded samples.

Access Control

  • Both verifications must pass to unlock the door.
  • Two failed attempts trigger an intrusion alarm.

Tech Stack

  • Face Recognition: OpenCV
  • Backend Processing: Flask
  • Styling: SCSS
  • Data Storage: PostgreSQL

Algorithms

  • Face Detection: Haar Cascades
  • Face Recognition: LBPH

Conclusion

SecureEntry will redefine home security by integrating facial and voice recognition for authentication. This system will ensure maximum security with seamless user access.
